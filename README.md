# Earth: Dolphins
## Solution Summary

The purpose of this repository is to provide a sample that illustrates the use of PlotStax® code generation.

In this sample, instead of going through the nuts and bolts of code generated by PlotStax, the sample focuses on some ways to build on top of the generated application structure.  The code-generation has already been done in this sample and the .NET Aspire solution can be run. (And code generation can of course be redone with different configurations for experimentation.)

## Dev Machine Dependencies / Prerequisites

* .NET 10 and .NET Aspire SDK version 13 or higher using Visual Studio 2026. (the free Community Edition suffices)
* Some basic familiarity with .NET Aspire, C#, and .NET will help in understanding what PlotStax does.
* NPM needs to be installed.

## DolphinUi

When running the solution the .NET Aspire dashboard includes a generated UI project.  If you click that entry in the dashboard it will open a page that has some custom code added to it for a LitElement demonstrating use of a generated proxy model for calling a service and another proxy model for subscribing to a broadcast.

## Dev-Managed code

All the non-generated changes made to files in this solution have a comment with the text "DEV_MANAGED_CODE_EXAMPLE" to make them searchable for further review.
